# Next Steps & Resources

This page provides curated links and references to advanced guidance, official documentation, and essential resources to help you smoothly transition to deeper adoption of GoogleMock and explore more challenging testing scenarios.

---

## 1. Advanced Guides and User Flows

Explore in-depth recipes and usage patterns for mastering mock creation, expectation setting, action delegation, and verification.

- **[gMock Cookbook](https://google.github.io/googletest/gmock_cook_book.html)**  
  _Practical recipes covering common and advanced mocking scenarios._  
  Includes guidance on mocking non-virtual methods, delegating calls to fakes and real objects, asynchronous testing strategies, matcher compositions, and more.

- **[gMock for Dummies](https://google.github.io/googletest/gmock_for_dummies.html)**  
  _Beginner-friendly, narrative-driven introduction to using gMock._  
  Walks you through mock class creation, using matchers and actions, setting expectations, and best practices for test maintainability.

- **[Legacy gMock FAQ](https://google.github.io/googletest/gmock_faq.html)**  
  _Answers longstanding questions and clarifies common pitfalls._  
  Helpful insights on virtual method requirements, warnings, heapcheck failures, behavior ordering, and interaction nuances.

- **[gMock Cheat Sheet](https://google.github.io/googletest/gmock_cheat_sheet.html)**  
  _Quick reference for common gMock constructs._  
  Summarizes mocking syntax, modifiers, and typical expectations setup.

## 2. Core Documentation Portal

The official documentation hub offers everything from API references to conceptual explanations.

- **[Mocking Reference](https://google.github.io/googletest/reference/mocking.html)**  
  Covers macros like `MOCK_METHOD`, `EXPECT_CALL`, `ON_CALL` with detailed argument explanations and usage examples.

- **[Actions Reference](https://google.github.io/googletest/reference/actions.html)**  
  Detailed documentation on built-in actions and how to define custom actions.

- **[Matchers Reference](https://google.github.io/googletest/reference/matchers.html)**  
  A comprehensive list of matchers, including how to create compound and custom matchers.
