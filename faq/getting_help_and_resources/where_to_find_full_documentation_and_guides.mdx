# Where Can I Find the Full Documentation and Additional Guides?

This page directs you to the comprehensive official **GoogleTest** documentation and highlights several useful guides to deepen your understanding of the framework or explore niche topics.

---

## Official Documentation and User Guides

For the fullest coverage of GoogleTest and GoogleMock, the primary resource is the official documentation hosted on GitHub Pages:

- **GoogleTest Documentation Homepage:** [https://google.github.io/googletest/](https://google.github.io/googletest/)

Here you will find high-quality, well-maintained, and detailed explanations including usage guides, API references, and conceptual overviews.

### Recommended Starting Points

- **[GoogleTest Primer](https://google.github.io/googletest/primer.html)** – Begin here to learn how to write simple and effective C++ tests.
- **[GoogleMock for Dummies](https://google.github.io/googletest/gmock_for_dummies.html)** – A friendly introduction to mocking with gMock, ideal for beginners.
- **[GoogleTest FAQ](https://google.github.io/googletest/faq.html)** – Answers to frequently asked questions about installation, usage, and troubleshooting.

### Guides and Cookbooks

To advance your skills or find solutions to specific challenges, consult these invaluable in-depth guides:

- **[gMock Cookbook](https://google.github.io/googletest/gmock_cook_book.html)** – Practical recipes and patterns for mocking, ideal for honing your gMock usage.

- **[Advanced Testing Patterns Guide](https://google.github.io/googletest/advanced.html#advanced_testing_patterns)** – Explore patterns like parameterized tests, death tests, and custom assertions.

- **[Best Practices for Reliable and Maintainable Tests](https://google.github.io/googletest/integration-and-best-practices/best-practices.html)** – Learn to write tests that are robust, easy to maintain, and scalable.

- **[Integration & Compatibility](https://google.github.io/googletest/overview/architecture-key-concepts/integration-with-build-and-tools.html)** – Understand how GoogleTest fits with build tools like CMake and Bazel for seamless integration.
