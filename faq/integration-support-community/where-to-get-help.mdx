# Where can I get support or ask further questions?

This page guides you to the official channels where you can get help, ask questions, and engage with the GoogleTest community. It summarizes the appropriate avenues for different support needs and sets clear expectations about response times and collaboration.

---

## Official Documentation

The best starting point is the comprehensive documentation available online:

- **GoogleTest User's Guide:** Covers everything from basic test writing to advanced topics.
- **GoogleTest Primer:** A step-by-step tutorial for beginners.
- **Advanced GoogleTest Topics:** In-depth material on specialized features and configurations.
- **API Reference:** Detailed descriptions of GoogleTest classes, macros, and functions.

Find the full documentation at [https://google.github.io/googletest/](https://google.github.io/googletest/).

---

## Community Forums and Discussions

Engage with other users and contributors for advice, troubleshooting, and best practices:

- **GitHub Discussions:** 
  - [https://github.com/google/googletest/discussions](https://github.com/google/googletest/discussions)
  - Use Discussions for questions, feature requests, and general conversations.
- **Stack Overflow:** Tag your questions with `googletest` to reach a broad developer audience.

When posting questions:

- Provide detailed descriptions of your problem.
- Include minimal, reproducible code snippets if relevant.
- Mention your GoogleTest version and environment.

---

## Issue Tracker

For bug reports or to track ongoing issues:

- Use the [GoogleTest GitHub Issues](https://github.com/google/googletest/issues) page.
- Before submitting a new issue, search existing issues to avoid duplicates.
- Follow the issue template carefully to include logs, error messages, and reproduction steps.

**Note:** The issue tracker is not intended for general usage questions.

---

## Chat and Real-Time Support

While GoogleTest contributors do not provide dedicated real-time support, some community channels exist for quick discussions and informal help:

- **GoogleTest Community Slack/Discord (Unofficial):** You may find community-run groups where users exchange tips and guidance. Links can often be found through GitHub discussions or external community websites.

Remember these channels are not official GoogleTest support and response times may vary.

---

## When to Use Each Support Avenue

| Support Type               | When to Use                                             | Expected Response Time                         |
|----------------------------|--------------------------------------------------------|-----------------------------------------------|
| Official Docs              | Learning, reference, and troubleshooting common issues | Immediate (self-service)                      |
| GitHub Discussions        | Questions, feature requests, and community advice      | Hours to days                                 |
| GitHub Issues             | Reporting bugs or feature regressions                   | Days, depending on prioritization             |
| Stack Overflow           | Community help, usage questions                         | Hours to days                                 |
| Chat/Community Channels   | Real-time informal help                                 | Variable, no guarantee                         |

---

## Best Practices for Seeking Help

- **Search first:** Review documentation and previous questions before posting.
- **Be specific:** Describe the exact problem, environment, and steps tried.
- **Provide code snippets:** Share minimal reproducible examples.
- **Be patient and courteous:** Open-source projects are often supported by volunteers.
