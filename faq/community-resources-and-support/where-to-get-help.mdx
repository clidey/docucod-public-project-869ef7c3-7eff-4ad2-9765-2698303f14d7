# Where to Get Help

This page guides GoogleTest and GoogleMock users on where to find support and assistance. It lists official resources, documentation portals, and vibrant community forums to help you get answers quickly and effectively.

---

## Official Support Channels

GoogleTest and GoogleMock maintain a few authoritative channels where users can seek formal support and access comprehensive documentation.

### 1. GoogleTest & GoogleMock Official Documentation

- **GoogleTest Primer and User Guides:** Start here for practical walkthroughs, tutorials, and detailed explanations covering basic to advanced testing concepts.
- **Reference Documentation:** API details for assertions, test macros, mocking, expectations, and actions.

These resources provide definitive answers and reduce troubleshooting time:

- [GoogleTest Primer](https://google.github.io/googletest/primer.html)
- [GoogleTest Advanced Topics Guide](https://google.github.io/googletest/advanced.html)
- [GoogleTest Reference - Assertions](https://google.github.io/googletest/reference/assertions.html)
- [GoogleMock Reference - Mocking](https://google.github.io/googletest/gmock/reference/index.html)

### 2. Source Repository

If you want to dive even deeper, the official GitHub repository hosts the entire codebase and documentation, including README files with build and installation instructions:

- [GoogleTest on GitHub](https://github.com/google/googletest)

Using the repository, you can:

- Explore the latest features and fixes
- Report bugs and request features
- Review sample tests and examples

### 3. Mailing Lists and Issue Trackers

The GoogleTest project’s GitHub issues page and community mailing lists (if available) are the best places to report bugs and get official feedback.

- [GitHub Issues for GoogleTest](https://github.com/google/googletest/issues)

Note: For best results, include minimal reproducible examples and environment details.

## Community Forums and Discussion

When you want fast answers or want to explore practical tips, the GoogleTest user community is an invaluable resource.

### 1. Stack Overflow

GoogleTest and GoogleMock are active tags on Stack Overflow:

- [GoogleTest tag on Stack Overflow](https://stackoverflow.com/questions/tagged/googletest)
- [GoogleMock tag on Stack Overflow](https://stackoverflow.com/questions/tagged/googlemock)

You can:

- Search for existing questions and answers
- Post your own questions with detailed descriptions
- Benefit from community-driven insights and solutions

### 2. Google Groups or Mailing Lists

Though less common nowadays with GitHub and Stack Overflow, some GoogleTest users still maintain mailing lists or Google Groups. Check the official website or GitHub for recommended lists.

### 3. Developer Chat and Social Platforms

Occasionally, GoogleTest contributors and users engage on platforms like Gitter, Slack rideshare groups, or Reddit. These informal venues provide real-time help but may not always be officially monitored.

## Tips for Effective Help-Seeking

When reaching out for help, keep the following tips in mind to get quality and timely responses:

- **Include Minimal Reproducible Examples:** Share small compilable snippets illustrating your problem.
- **Specify Your Environment:** Include OS, compiler version, build system details, GoogleTest version.
- **Search Before Posting:** Use targeted keywords reflecting your error message or scenario.
- **Check for Similar Issues:** Many common problems have documented solutions.
- **Be Clear and Concise:** Describe expected vs actual behavior.
