# Official Documentation and Community Channels

The GoogleTest and GoogleMock frameworks are supported by a vibrant community and complemented by a rich set of official documentation. To ensure you always have access to the latest knowledge, user guides, FAQs, and community resources, this page directs you to key channels and materials.

---

## Official GoogleTest and GoogleMock Documentation

- **GoogleTest and GoogleMock GitHub Repository:**
  The main source code repository hosts the freshest releases, issues, pull requests, and examples.

  [https://github.com/google/googletest](https://github.com/google/googletest)

- **GoogleTest and GoogleMock GitHub Pages:**
  The official user guides, API references, FAQs, and tutorials are published here with navigable structure and search capabilities.

  [https://google.github.io/googletest/](https://google.github.io/googletest/)

- **Frequently Consulted Guides:**

  - [gMock FAQ](https://google.github.io/googletest/gmock_faq.html): Answers to common questions and troubleshooting tips.
  - [gMock Cookbook](https://google.github.io/googletest/gmock_cook_book.html): Practical recipes and usage patterns.
  - [Mocking Reference](https://google.github.io/googletest/reference/mocking.html): Detailed syntax and API documentation.
  - [Matchers API](https://google.github.io/googletest/reference/matchers.html): Guide to argument matchers.
  - [Actions API](https://google.github.io/googletest/reference/actions.html): Details on mock function behaviors.
  - [Cardinalities API](https://google.github.io/googletest/reference/cardinalities.html): Controlling call counts and expectations.

- **Getting Started Guides:**

  Structured primers to ramp you up quickly:

  - [gMock for Dummies](https://google.github.io/googletest/gmock_for_dummies.html)
  - [Creating and Using Mocks](https://google.github.io/googletest/guides/mocking-best-practices/creating-mocks.html)
  - [Writing Expectations and Actions](https://google.github.io/googletest/guides/mocking-best-practices/writing-expectations-actions.html)
