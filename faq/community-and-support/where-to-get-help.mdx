# Where Can I Get Help?

This page provides you with comprehensive information about the various support channels available for GoogleTest and GoogleMock. Whether you're encountering issues, seeking guidance, or want to contribute, this guide helps you find quick assistance and reliable official responses.

---

## Recommended Support Channels

### Mailing Lists

- Join the GoogleTest mailing lists to discuss issues, ask questions, and share feedback with the GoogleTest community and developers.
- This is an excellent way to get advice from other users and the project maintainers.

### Issue Trackers

- Report bugs or request features via the official GitHub issue tracker at the [GoogleTest repository](https://github.com/google/googletest/issues).
- Before reporting a bug, search existing issues to avoid duplicates.
- Provide detailed information including reproducing steps, environment, and relevant code snippets to get effective support.

### Forums and Community Resources

- Use community forums, Stack Overflow, and Google groups dedicated to testing and GoogleTest topics.
- The community is active and can offer practical tips and alternate solutions.
- Tag your posts clearly, for example, with `googletest` or `gmock`, to attract expert attention.

---

## Bug Reporting Guidelines

To help the developers address issues efficiently, please:

- Use the official GitHub issue tracker.
- Provide a minimal, complete, and verifiable example to reproduce the problem.
- Include version information for GoogleTest/GoogleMock, compiler, OS, and build tools.
- Clearly describe expected behavior vs. observed behavior.
- Attach logs or error output if applicable.

Avoid reporting bugs in general discussion forums to ensure your report reaches developers directly.

---

## Feature Requests

If you want to suggest enhancements or new features:

- Use the GitHub issue tracker to submit a feature request.
- Explain the use case and benefits clearly.
- Reference any relevant existing issues or pull requests.

Proposals with clear motivation and practical examples have a greater chance of being considered.
