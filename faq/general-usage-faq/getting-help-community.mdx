## Reporting Issues and Requests

If you encounter bugs, want to request features, or propose improvements, please use the GitHub issue tracker of the GoogleTest project:

- GitHub Repository: [https://github.com/google/googletest/issues](https://github.com/google/googletest/issues)

**Best Practices:**
- Search existing issues before opening a new one to avoid duplication.
- Provide detailed descriptions, including your environment setup, version, minimal reproducible examples, and error messages.

## Tips for Effective Support Requests

- Clearly describe what you're trying to achieve and what your current approach is.
- Include relevant code snippets where possible.
- Specify your development environment (compiler, OS, GoogleTest version).
- Indicate what you have tried and the exact nature of the issue or confusion.
