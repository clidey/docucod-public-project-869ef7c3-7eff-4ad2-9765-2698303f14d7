# Release Overview & Timeline

This page presents a clear and chronological summary of the major and minor releases for **GoogleTest** and **GoogleMock**. It is designed to help you quickly scan through recent updates, recognize major version milestones, and easily navigate to detailed release notes for specific versions.

---

## Purpose

As a developer or user of GoogleTest and GoogleMock, you want swift access to the history and highlights of new features, bug fixes, and changes in each release. This page consolidates essential release information in one place, serving as your go-to reference for understanding the evolution of the frameworks.

---

## How to Use This Page

- **Scan Recent Releases:** Reviews the most current versions with succinct summaries of new capabilities and important fixes.
- **Identify Major Versions:** Detect major milestone releases that may introduce significant new features or API changes.
- **Access Detailed Notes:** Follow direct links to comprehensive release notes for each version.

This workflow supports rapid evaluation and decision-making about upgrading or exploring new functionality within the GoogleTest & GoogleMock ecosystem.

### Latest Release

The most recent release is prominently shown at the top to spotlight the newest changes you should be aware of.

### Release Highlights

Each release entry includes:

- **Version Number**: Clearly noted to distinguish point, minor, and major updates.
- **Release Date:** When the version was made available.
- **Key Features:** Summary of added functionalities and improvements.
- **Bug Fixes:** Notable corrections that enhance stability or correctness.
- **Security Updates:** Patches addressing vulnerabilities.
- **Links to Detailed Notes:** Directs you to full release documentation.

### Example Entry
For illustration, a typical release entry looks like this:

---

**Version 1.17.0** | Released June 10, 2024

- Introduced mandatory C++17 requirement for the 1.17.x branch.
- Enhanced mocking flexibility and test discovery mechanisms.
- Improved error message clarity with new predicate-based assertions.
- Fixed several flaky test cases and threading issues.
- Security improvements in exception handling.
- [View detailed release notes](https://github.com/google/googletest/releases/tag/v1.17.0)
