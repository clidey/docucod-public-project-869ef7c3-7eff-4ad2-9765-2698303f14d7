# Latest Release Overview

This page delivers a focused summary of the most recent GoogleTest and GoogleMock release, tailored for users seeking a quick, practical glance at current enhancements. It highlights the version number, top headline features, notable bug fixes, and provides direct access to the full release notes for deeper exploration.

## 1. Release Version and Date

- **Version:** 1.17.0
- **Release Date:** See the [official release page](https://github.com/google/googletest/releases/tag/v1.17.0) for details.


## 2. Headline Features

This release advances GoogleTest and GoogleMock with important updates designed to improve testing experience, integration, and standards compliance:

- **Minimum C++ Standard Updated:**
  The 1.17.x branch now requires at least C++17, enabling the use of modern language features and improving compiler compatibility.

- **Dependency Plans:**
  A planned future update will introduce dependencies on [Abseil](https://github.com/abseil/abseil-cpp).

- **Enhanced Mocking with GoogleMock Included:**
  GoogleMock remains fully integrated and benefits from improvements in the latest release.

- **Documentation on GitHub Pages:**
  Updated and hosted documentation at [GoogleTest GitHub Pages](https://google.github.io/googletest/) provides improved browsing and user experience.


## 3. Notable Bug Fixes and Improvements

- Continuous integration stability improvements, leveraging Google's internal CI pipelines.
- Enhanced compiler and platform support ensuring smoother builds on Linux, Windows, and macOS.
- Fixes addressing known issues around threading, build flags, and mock lifecycle management.


## 4. Upgrade and Migration Notes

Upgrading to version 1.17.0 involves the following considerations:

- **C++17 Minimum Requirement:**
  Verify your build environment supports at least C++17 standard to avoid compatibility issues.

- **Build System Compatibility:**
  Confirm your CMake, Bazel, or custom build scripts accommodate new version and flag requirements.

- **Review Deprecations and Breaking Changes:**
  Although this summary emphasizes new features and fixes, consult the [breaking changes documentation](https://github.com/google/googletest/releases/tag/v1.17.0) and migration guides where applicable.
