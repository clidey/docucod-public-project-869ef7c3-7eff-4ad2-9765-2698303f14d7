## Sample Entry: GoogleTest 1.17.0

### Release Date: February 15, 2023

- **Highlights:**
  - GoogleTest 1.17.x branch now requires **C++17** or later, aligning with modern C++ standards for better language features and performance.
  - Improved **build system support** to streamline integration on varied platforms.
  - Enhanced **multithreading capabilities** for more reliable concurrent test execution.
- **Documentation:** Visit the [GoogleTest User's Guide](https://google.github.io/googletest/) for updated best practices and examples compatible with this release.
- **GitHub Release:** [v1.17.0 on GitHub](https://github.com/google/googletest/releases/tag/v1.17.0)
