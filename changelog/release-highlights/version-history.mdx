# Version History

This page provides a comprehensive chronological list of all GoogleTest releases, including GoogleMock changes, with their release dates and concise summaries. It is designed to help users and teams quickly understand the evolution of the GoogleTest and GoogleMock projects, facilitating awareness of new features, improvements, and fixes over time. This historical perspective aids in evaluating upgrade cadence, planning updates, and identifying when specific capabilities or changes were introduced.

---

## How to Use This Version History

When maintaining or upgrading your testing suite with GoogleTest and GoogleMock, understanding the history of releases is crucial. This page enables you to:

- Quickly find what was added, changed, or fixed in a particular version.
- Assess upgrade timing based on new features or important fixes.
- Understand when breaking changes or deprecations occurred (refer to the breaking changes and deprecation notices pages for detailed migration instructions).

Use this list as your go-to reference to monitor trends, detect improvements relevant to your use cases, and align your testing infrastructure with GoogleTest's official support and feature roadmap.

---

## Release Overview

Each release entry is organized as follows:

- **Version Number and Release Date:** Identifies the precise version and its official release date.
- **Summary:** A clear and concise description highlighting significant changes, new features, critical fixes, or important notes.

This structured approach promotes clarity and ease of scanning.

## Recent Releases

### Version 1.17.0 - Released April 4, 2023
- Introduced C++17 as the minimum required standard for compilation, enhancing code quality and modern language feature use.
- Integrated improvements in GoogleMock for more expressive mock behaviors.
- Announced upcoming dependency on Abseil for better utilities support.

### Version 1.16.0 - Released September 2022
- Added new assertions to simplify common testing scenarios.
- Improved test discovery performance.
- Enhanced platform compatibility and bug fixes targeting Windows and Linux builds.

### Version 1.15.0 - Released March 2022
- Upgraded mocking APIs to support advanced matchers and actions.
- Refined death test reliability under multithreaded conditions.
- Fixed notable issues reported in test execution reporting.

---

## Legacy Releases

### Version 1.14.0 - Released December 2021
- Supported additional compiler toolchains.
- Added granular mock strictness controls.

### Version 1.13.0 - Released August 2021
- Introduced parameterized and typed test enhancements.
- Increased platform portability for embedded use cases.

### Version 1.12.0 - Released April 2021
- Fixed critical bugs impacting mock object lifecycle.
- Improved XML output formatting for CI integration.
